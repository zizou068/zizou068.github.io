## 💡 FAQ

### Q1: What is MAVROS?

MAVROS is a communication library that connects the Robot Operating System (ROS) with the MAVLink protocol used in drone communications.

### Q2: Can I use this software on a Raspberry Pi?

Yes, this software is designed to run on Raspberry Pi 4, allowing you to manage flights directly from a compact device.

### Q3: Is it necessary to have programming knowledge to use this software?

No, this application is designed for users without any programming knowledge. The installation and usage process is straightforward, with clear guidance.
